# Создатель Git
![Линус Торвальдс](/GitHistory/303c030a5146844e.png)

Линус Бенедикт Торвальдс (род. 28 декабря 1969)
Финно-американский программист, создатель ядра Linux и системы контроля версий Git

# Основные исторические этапы
2005 — Рождение Git
Апрель 2005: Начало разработки после потери доступа к BitKeeper

7 апреля 2005: Первый коммит в Git

16 апреля 2005: Git становится самодостаточным

25 июля 2005: Первый публичный релиз Git 0.99

Предыстория:
До 2005 года ядро Linux использовало проприетарную систему BitKeeper. После конфликта лицензий Линус Торвальдс создал собственную распределённую систему контроля версий.

2006-2007 — Раннее развитие
Быстрая итерация версий

Формирование основных концепций

Привлечение первых сторонних разработчиков

2008 — Появление GitHub
Основание GitHub Томом Престон-Вернером

Git получает удобный веб-интерфейс

Начало "социального программирования"

2010 — Поворотный момент
Git превосходит SVN по популярности

Становится стандартом в open-source сообществе

2014 — Enterprise-внедрение
Microsoft объявляет о переходе на Git для разработки Windows

Крупные компании начинают массово внедрять Git

2018 — Покупка GitHub
Microsoft покупает GitHub за $7.5 млрд

Признание стратегической важности экосистемы

2020-настоящее время — Доминирование
95% разработчиков используют Git

# Стандарт де-факто в индустрии

Интеграция с современными DevOps-практиками

Технические преимущества:
Распределённая архитектура — полная копия репозитория у каждого разработчика

Высокая производительность — большинство операций локальные

Надёжность — SHA-1 хеширование для обеспечения целостности данных

Гибкое ветвление — лёгкое создание и слияние веток

Социальные факторы:
GitHub сделал Git доступным и социальным

Pull Request как новый способ сотрудничества

Сообщество open-source приняло Git как стандарт
